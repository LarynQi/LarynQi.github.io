Slowify

Play music however you like with the official AFX Music App. Upload your own music and share with your friends. Perfect for dancers, musicians, or singers alike!

AFX is a dance group at UC Berkeley. As computer science students who love dance, AFX Tech Committee built this app to serve not only our dance community at Cal but also dancers and artists all around the world. Features include looping, speed adjustment, visualization, and link sharing with others. The Android app is also designed to be compatible with the website version.

  • Organization: AFX Tech
  • Tech: Android Studio • Java • MediaPlayer API
  • Link: Google Play Store
  • Date: Fall 2019 - Spring 2020

Gitlet

A version-control system that mimics some of the basic features of the popular system Git.

Unlike other class projects, Gitlet required that each student design their own implementation and build it from scratch—no skeleton code. My implementation utilized persistence, cryptographic hashing, a variety of data structures, and graph traversals. Beyond the techincal intricacies, this project emphasized software engineering design principles such as efficiency, decomposition, and clean style. Unfortunately, the design doc and code cannot be publicly shared as per course guidelines.

  • Organization: UC Berkeley CS61B (Data Structures)
  • Tech: Java
  • Link: Spec
  • Date: April 2020

uMaps

An innovative indoor mapping iOS app designed to be fully integrated with Google Maps.

uMaps tackled an unsolved problem in the geospatial realm: indoor navigation. While other team members worked with Swift and Google Maps API, I focused on the backend. I implemented an object-oriented approach to representing building data and used the A* algorithm to efficiently find paths. Additional functionality included accessbility features and multi-floor capability.

  • Organization: TreeHacks
  • Tech: Python • FirebaseDB
  • Link: Presentation, Repo
  • Date: February 2020

Camping VR

CampingVR was the final project of the class create a camping simulator with game mechanics similar to Minecraft and 5 main quests to complete.

Music Visualizer

This Music Visualizer takes in any mp3 file and visualizes the Fast Fourier Transform and how it changes over time using 512 cubes in a ring and 8 bins in the center that represent the primary frequency bands.
